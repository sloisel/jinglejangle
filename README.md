# Jingle Jangle

A Flutter app for learning Japanese Hiragana and Katakana through interactive games.

**[Play Now](https://sloisel.github.io/jinglejangle/)**

## Features

- **Tile Matching Game**: Tap falling tiles that match the spoken syllable
- **Spelling Mode**: Type out words using an on-screen keyboard
- **Native Audio**: All Hiragana and Katakana syllables recorded by a native speaker
- **Adaptive Learning**: The game tracks your progress and focuses on characters you find difficult

## Supported Character Sets

- Hiragana (basic + dakuten + combinations)
- Katakana (basic + dakuten + combinations)

## Installing on iPad/iPhone

You can install Jingle Jangle as a web app on your home screen:

1. Open **[sloisel.github.io/jinglejangle](https://sloisel.github.io/jinglejangle/)** in Safari
2. Tap the **Share** button (square with arrow pointing up)
3. Scroll down and tap **Add to Home Screen**
4. Tap **Add** in the top right corner

The app will now appear on your home screen and run in full-screen mode like a native app.

## Running Locally

```bash
flutter pub get
flutter run
```

## Building for Web

```bash
flutter build web
```
